# Inventory Frontend

React + Vite frontend for the `MSE-1` inventory backend.

## Features
- Add product
- View all products
- Edit product
- Delete product
- Search by product name
- Filter by category

## Run
1. Install packages:
`npm install`

2. Optional environment file:
Create `.env` from `.env.example`

3. Start the frontend:
`npm run dev`

The Vite dev server proxies `/api` calls to `http://localhost:5000` by default.
